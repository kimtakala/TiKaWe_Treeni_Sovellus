# TiKaWe Treeni Sovellus

TiKaWe Treeni Sovellus on kuntosaliharjoitteluun keskittyvä web-sovellus. Sovelluksessa käyttäjä voi hallita liikkeitä ja niistä koostuvia treenejä, tarkastella muiden lisäämiä sisältöjä sekä seurata omaa harjoitteluaan.

## Sovelluksen keskeiset toiminnot

- Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
- Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan liikkeitä.
- Käyttäjä pystyy luomaan liikkeistä koostuvia treenejä.
- Käyttäjä näkee sovellukseen lisätyt liikkeet ja treenit.
- Käyttäjä pystyy etsimään liikkeitä hakusanalla.
- Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja sekä käyttäjän lisäämät liikkeet ja treenit.
- Käyttäjä pystyy valitsemaan liikkeelle yhden tai useamman luokittelun, kuten lihasryhmän.
- Käyttäjä pystyy arvioimaan liikkeitä ja treenejä asteikolla 1-5.

## Tietokohteet

Pääasialliset tietokohteet ovat liikkeet ja treenit. Toissijainen tietokohde on arvio, jonka käyttäjä voi liittää liikkeeseen tai treeniin.
