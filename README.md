Sovellus on kuntosali treeni sovellus, jossa on seuraavat toiminnot:

- Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
- Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan liikkeitä.
- Käyttäjä pystyy luomaan liikkeistä koostuvia treenejä.
- Käyttäjä näkee sovellukseen lisätyt liikkeet.
- Käyttäjä pystyy etsimään liikkeitä hakusanalla.
- Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja ja käyttäjän lisäämät liikkeet ja treenit.
- Käyttäjä pystyy valitsemaan ilmoitukselle yhden tai useamman luokittelun (lihasryhmät).
- Käyttäjä pystyy arvioimaan liikkeitä ja treenejä. (1-5)

Tässä pääasialliset tietokohteet ovat treenit ja liikkeet, toissijainen tietokohde on arviot.
